# 使い方
## 1. 準備
OSCのtypeで"PS01"を選択した後、以下の設定を行ってください。
- OSC
  - LFO depth : S100
- EG
  - Type : Open
- ARP
  - On/Latch
  - Arpegglator setting : Octave (= 起動時の設定)

## 2. PS01の遊び方
以下のパラメータを変更してください
- OSC
  - A knob (アルペジェレーターの範囲)
  - B knob (リリースタイム)
  - parameter 1 (コードの種類)
  - LFO rate (アルペジェレーターのパターンに影響)
- ARP
  - ARP pattern length (アルペジェレーターのパターンに影響)
